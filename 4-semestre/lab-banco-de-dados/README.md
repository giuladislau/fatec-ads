# Laboratório de Banco de Dados

Atividades e experimentos desenvolvidos na disciplina **Laboratório de Banco de Dados**.

## Objetivo da disciplina

Apresentar, de forma prática, ferramentas e tecnologias utilizadas no desenvolvimento de aplicações que utilizam banco de dados.
A disciplina tem foco em projetos experimentais e na familiarização com tecnologias utilizadas no mercado.

## Conteúdos trabalhados

* Conceitos básicos de banco de dados
* Criação e manipulação de tabelas
* Consultas SQL
* Integração entre aplicações e banco de dados
* Estrutura básica de APIs
* Desenvolvimento de pequenas aplicações para prática

## Tecnologias utilizadas

* Node.js
* Express.js
* DBeaver
* HTML
* CSS
* SQL

## Observação

Durante a disciplina são desenvolvidos diversos pequenos projetos com o objetivo de experimentar diferentes ferramentas e compreender como aplicações interagem com bancos de dados.
