# Linguagem de Programação

Atividades desenvolvidas na disciplina **Linguagem de Programação**.

## Objetivo da disciplina

Aprofundar o uso da linguagem C, explorando conceitos mais avançados de programação e estruturação de código.

## Conteúdos trabalhados

* Estrutura de programas em C
* Funções
* Vetores e matrizes
* Manipulação de strings
* Ponteiros
* Estruturas (struct)
* Organização de código
* Resolução de problemas utilizando C

## Linguagem utilizada

* C

## Observação

Este diretório contém exercícios e pequenos programas desenvolvidos durante as aulas para praticar os conceitos da linguagem.
