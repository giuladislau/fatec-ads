# Programação Web

Repositório de exercícios e pequenos projetos desenvolvidos na disciplina **Programação Web**.

## Objetivo da disciplina

Apresentar os conceitos fundamentais de desenvolvimento web utilizando PHP no lado do servidor, integrando páginas web com processamento de dados e banco de dados.

## Conteúdos trabalhados

* Introdução ao PHP
* Sintaxe básica
* Variáveis e estruturas de controle
* Formulários HTML com PHP
* Manipulação de dados enviados por formulários
* Integração com banco de dados
* Estrutura básica de aplicações web

## Tecnologias utilizadas

* PHP
* HTML
* CSS
* SQL

## Observação

Os arquivos presentes nesta pasta representam atividades acadêmicas e exercícios de prática realizados durante o curso.
