# Algoritmos e Lógica de Programação

Exercícios e atividades da disciplina **Algoritmos e Lógica de Programação**, utilizando a linguagem C como ferramenta para desenvolver o raciocínio lógico.

## Objetivo da disciplina

Desenvolver o pensamento lógico e a capacidade de resolver problemas computacionais por meio da construção de algoritmos.

## Conteúdos trabalhados

* Conceitos de algoritmo
* Estrutura básica de programas em C
* Variáveis e tipos de dados
* Operadores
* Estruturas condicionais
* Estruturas de repetição
* Vetores
* Funções
* Resolução de problemas com algoritmos

## Linguagem utilizada

* C

## Observação

Os códigos representam exercícios práticos realizados para fixação dos conceitos de lógica de programação.
